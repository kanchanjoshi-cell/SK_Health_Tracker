🌿 SK_Health_Tracker
A personalized, responsive web application engineered to calculate and manage custom diet structures, workout adaptations, and vital medical logs based on individual user goals.
🎯 Overview
SK_Health_Tracker is a comprehensive, client-side digital health companion designed to bridge the gap between dynamic fitness targets and basic medical screening. Whether a user is focused on healthy mass acceleration (bulking from a baseline like $40\text{ kg}$ to a target $52\text{ kg}$), targeted weight reduction, or cardiovascular monitoring, the application dynamically mutates its algorithms to provide tailored macroscopic dietary baselines.The platform prioritizes adaptive nutrition, ensuring that foundational macro-splits (Protein, Carbohydrates, and Fats) scale accurately against raw physical data using verified metabolic formulas.
✨ Key Features
Dynamic Onboarding & Goal Adaptation: Computes baseline energy requirements using the Mifflin-St Jeor equation and instantly recalibrates targets based on specified caloric surpluses or deficits.Macronutrient Scaling: Automatically calculates optimal distributions of Carbs, Proteins, and Fats optimized for target weight achievement.Everyday Vitality & Medical Logs: Dedicated monitoring tracks daily weight fluctuations alongside systemic blood pressure ($120/80\text{ mmHg}$) to ensure safe physiological adaptation.Intuitive UX Architecture: Built with a modern, high-contrast, slate-dark dashboard designed to reduce visual strain during everyday entry logging.
🛠️ Technical Stack
Structure: Semantic HTML5Styling: Modern CSS3 (utilizing CSS Grid, Flexbox layouts, and custom :root CSS variables)Logic: Native Vanilla JavaScript (ES6+)
