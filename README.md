SK_Health_Tracker is a comprehensive, client-side digital health companion designed to bridge the gap between dynamic fitness targets and basic medical screening. Whether a user is focused on healthy mass acceleration (bulking from a baseline like $40\text{ kg}$ to a target $52\text{ kg}$), targeted weight reduction, or cardiovascular monitoring, the application dynamically mutates its algorithms to provide tailored macroscopic dietary baselines.The platform prioritizes adaptive nutrition, ensuring that foundational macro-splits (Protein, Carbohydrates, and Fats) scale accurately against raw physical data using verified metabolic formulas.
